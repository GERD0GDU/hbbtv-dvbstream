Opencaster-Sample (Adapted for Windows Visual Studio 2013)
==================================================

OpenCaster is a free and open source MPEG2 transport stream data generator and packet manipulator.

## Includes

* esaudio2pes
* esvideompeg2pes
* pesaudio2ts
* pesvideo2ts
* sec2ts
* tscbrmuxer
